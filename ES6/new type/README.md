
# new type


## Decription

- let과 const 키워드 추가

  

## Types

>  var 
>- 변수의 재선언 가능
>- 재할당 가능 
>- 범위는 function
>- 호이스팅 가능

> let 
>- 변수의 재선언 불가
>- 재할당 가능
>- 범위는 block{}
>- 호이스팅 불가

>const
>- 상수로 취급함으로써 변수의 재선언
>- 재할당 불가
>- 범위는 block{}
>- 호이스팅 불가

## Hoisting
- 변수를 유효 범위의 최상단으로 끌어 올리는 행위
- 변수의 정의가 범위에 따라 선언과 할당으로 분리
- 함수의 경우 선언식의 경우에만 호이스팅되며 표현식은 에러를 발생시킴
